## 3.1. Ежемесячные продажи по сегментам

select extract(year from order_date) as year,
 extract(month from order_date) as month,
 segment,
 round(sum(sales), 2) as monthly_sales
from orders
group by year, month, segment
order by 1, 2, 3;
